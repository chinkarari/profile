Lesson 1 The Basics of the Web and HTML

the World Wide Web

There is three components of the web; the computer(browser), the Internet and the servers.The Browser makes requests via the Internet to the servers. We use a protocol http to get pages from servers.

HTML

HTML = "HyperText Markup Language"

HTML is made up of text content, markup to decide what it looks like, reference to other documents and links to other pages and the main contents on the web.

Tag

To markup the text, we use tags. The basic structure of HTML is an element which is structured <tag name> contents </tag name>. There are two kinds of tag; block and inline. Each tags define a block element and an inline element. Block elements take up the full available width and starts on a new line. Inline elements can occupy the same line and take up as much width as necessary.

block elements: <div>, <h1>-<h6>, <p>

inline elements: <span>, <a>, <img>

Lesson 2 Creating a Structured Document with HTML

HTML-CSS-DOM

A web page consists of HTML, CSS and DOM. HTML (HyperText Markup Language) gives the structure of a web page and the content of the page, CSS(Cascading Style Sheets) defines the style of the web page and DOM (Document Object Model) can access objects in HTML and the nodes in HTML are structured like a tree structure.

Box Structure

Every element in a web page is defined as a box and it's important to understand the concept of the box for a web developer. To define the box, we uses a div tag.When we add the style to boxes, set a class attribute like a label.

Lesson 3 Adding CSS Style to HTML Structure

What is CSS?

CSS stands for Cascading Style Sheets. Cascading means that rules apply every elements not only the element that CSS directly applied but also the all child' elements of it. If you want to add CSS style to a web page, there are three ways.

  • write CSS in the <head> tag
  • Link HTML to a CSS file
  • write style inline with HTML

When we define a specific element to add style, we use a HTML tag or a class attribute as a selector.

There are a lot of function in CSS, so it is useful to refer CSS reference when you add CSS style.

Semantic Tags

Semantic tags are tags with a meaning. To use semantic tags, the browser and the developer can easily understand what the element describes. Using semantic tags helps search engines index the content of a web page.

Box in CSS

All elements are structured as a box in HTML documents. The box consists of Margin, Border, Padding, Contents. We can set the box size in pixels and percentages.
